# Blogging Platform (Learning Project)

This project is a simple **blogging platform** built while learning full-stack development using **AI tools and YouTube tutorials**.

The goal of this project is to understand how frontend and backend applications work together.

---

## 🛠️ Technologies Used

Frontend

* React
* Tailwind CSS
* Vite

Backend

* Node.js
* Express.js
* MongoDB
